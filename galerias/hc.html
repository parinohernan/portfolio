Henrry Countries

Es una aplicación que muestra informacion de los distitos, tambien podemos crear actividades turisticas y asociarlas a cada pais.

  • Diseñar en React sin librerías externas y desarrollar frontend y backend de una aplicación web que muestra información de países.
  • Programar un servidor en Node creando API Rest en express que BFF con endpoints personalizados conectando a la base de datos postgres utilizando Sequelize ORM y trayendo datos crudos de la API countries . Y permita realizar CRUD de actividades turísticas.
  • Programar búsqueda, filtrado combinado y ordenamiento, utilizando Redux.
  • Desarrollar un componente utilizando formularios controlados para registrar actividades turísticas.

FrontEnd: React, Redux, CSS. LocalStorage.

Node, Express, Sequelize, Postgres.
