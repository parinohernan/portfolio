Proyecto Ge3kHub


Ge3kHub

Es un ecommerse de productos geek, podras encontrar articulos geniales y divertidos.

Tareas realizadas en el proyecto

  • Desarrollar y diseñar en React front end un ecommerce donde se venden productos Geek
  • Programar el envío de notificaciones a usuarios a través de Nodemailer colaborando en el backend
  • Dar estilos a los componentes usando Bootstrap.
  • Programar filtros combinados utilizando Zustand usando la lógica desde el FrontEnd
  • Desarrollar un carrito de compras, con pasarela de pagos en STRIPE, autenticación con cuentas de terceros, dashboard admin. Diseño de ERM, Investigar nuevas librerías como Auth0 y como GIT Flow.
  • Desarrollar componentes de profile y login permitiendo cuentas de terceros utilizando auth0 y establecer permisos para usuarios invitados, registrados y verificados.

FrontEnd: React, Bootstrap, Zustand, Aut0, Stripe, Git, SCRUM
